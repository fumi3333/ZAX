### 1. マリッジパクト（Marriage Pact）仕様の50問への完全移行
- **背景**: 以前に合意していた「マリッジパクトを参考にした50問の質問」へ移行する方針を、コンテキストウィンドウトランケーションにより失念していたため復元。

- **新しい5つのカテゴリ**: 
  1. Lifestyle (ライフスタイル)
  2. Values (価値観・社会)
  3. Romance (恋愛・親密さ)
  4. Conflict (コンフリクト解決・許容範囲)
  5. Ambition (野心・キャリア)
- 各カテゴリ10問、計50問の構成に変更。

### 2. 分析・レーダーチャートの次元再マッピング
- **実装内容**: `src/app/api/diagnostic/submit/route.ts` にて、50問から算出された5カテゴリのスコアを、レーダーチャート用の「6次元ベクトル」に変換するロジックを更新。
- **新しい6次元（レーダーチャートの頂点）**:
  1. 生活基盤 (Lifestyle)
  2. 社会意識 (Values)
  3. 親密性 (Romance)
  4. 対話力 (Conflict)
  5. 野心 (Ambition)

- `DiagnosticResultClient.tsx` および `CompareRadarChart.tsx` の表示ラベルも上記に更新。

### 3. UIの微調整とバグ修正（DiagnosticWizard）
- **質問番号の重複表示の解消**: 画面右下に表示されていた「x / 59 問回答済み」の文字を削除。カード上部の「質問 x / 50」のみに一本化。
- **レイアウトのズレ解消**: ラジオボタン横の「同意しない」「同意する」のテキストが折り返されて（同意し\nない等）縦にズレていた問題を解消。`whitespace-nowrap` と `items-center` を用いて、テキストが常に一行で美しく中央揃えになるように修正。
