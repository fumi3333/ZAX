# ZAX プロジェクト概要

**バージョン**: Phase 1 (MVP)  
**最終更新**: 2026-02-15  
**ステータス**: Hackathon Demo Ready

---

## 🎯 プロジェクトの本質

ZAX は、人間の内面的特性（価値観・思考パターン・行動傾向）を多次元ベクトル空間上の「座標」として捉え、数学的な距離計算によって「本質的に共鳴する相手」を発見するマッチングプラットフォームです。

**最終目標**: 長期的な幸福を最大化する、個人と集団の相互学習システム

---

## 🧠 コアコンセプト

### 1. 属性ではなく「本質」でマッチング

| 従来型マッチング | ZAX |
|---|---|
| 年齢・年収・見た目 | 価値観ベクトル |
| 静的プロフィール | 時系列で変化するアイデンティティ |
| 「似た人」を探す | 「成長を促す人」を探す |
| データは捨てられる | RAGで継続的に学習 |

### 2. 補完性スコアの設計思想

**仮説**: 類似度1.0（完全一致）はエコーチェンバーを生む

```
類似度 0.5 付近 = 共感できる + 新しい刺激
→ 成長を促す理想的な距離

補完性スコア = exp(-(similarity - 0.5)² / 0.1) × 100
```

### 3. RAG × 時系列データによる個人最適化

- マッチング履歴を時系列で蓄積
- pgvector で過去の類似体験を検索
- Gemini にコンテキストとして渡し、より精度の高い推薦を生成

## 🚀 主要機能（Phase 1 実装済み）

### 1. 50問性格・価値観診断

- 7段階バブルUIで直感的に回答
- カテゴリ: 論理性、直感力、共感性、意志力、創造性、柔軟性
- 自動進行（回答選択で次へ）

### 2. AI分析パイプライン

```
回答データ (50問)
  ↓
カテゴリ別スコア集計
  ↓
Gemini Pro で性格分析レポート生成
  ↓
text-embedding-004 で 768次元ベクトル化
  ↓
PostgreSQL + pgvector に保存
```

### 3. ベクトルマッチング（Top 5）

- pgvector の HNSW インデックスで高速検索
- 補完性スコアでランキング
- 50人の日本人名アーキタイプがフォールバック（DB未接続時）

### 4. マッチング結果UI

- グラスモルフィズムデザイン
- 上位3名をカード表示
- 比較レーダーチャート（自分 vs 相手）
- AI推薦理由の表示

### 5. フィードバック収集

- 4つのスライダー（1-10）:
  - 相手は楽しそうですか？
  - 自分は充足感を感じますか？
  - 成長できそうですか？
  - また会いたいですか？
- 自由記述欄

## 📈 データフロー（ユーザー視点）

```
1. ランディングページ訪問
   ↓
2. 「無料で診断を開始」クリック
   ↓
3. 50問診断（5-7分）
   ↓
4. AI分析中（15-20秒）
   ↓
5. 結果表示:
   - レーダーチャート（6次元）
   - AI分析レポート
   - マッチング候補 Top3
   ↓
6. 各候補に「フィードバックを送る」
   ↓
7. フィードバックデータを保存（将来の学習用）
```

## 🧪 技術的な挑戦

### 1. 補完性スコアの数学的妥当性

- ベルカーブによる成長ポテンシャル評価
- 心理学的仮説（エコーチェンバー回避）との整合性

### 2. RAGによるパーソナライゼーション

- ファインチューニング不要
- コスト効率的（文脈をプロンプトに追加するだけ）
- 少量データでも改善効果

### 3. pgvector の実践投入

- HNSW インデックスによる高速検索
- リレーショナルデータとベクトルデータの統合管理
- 外部サービス不要（コスト削減）
