Algorithm
補完性スコア アルゴリズム
growthScore  =  exp( −(similarity − 0.5)² / 0.1 )  ×  100
similarity ≈ 1.0
低スコア

完全一致
→ エコーチェンバー形成
→ 成長が止まる

similarity ≈ 0.5  🎯
最高スコア

適度な差異
→ 共感 + 新しい刺激
→ 最も成長を促す

similarity ≈ 0.0
低スコア

完全に無関係
→ 価値観の衝突
→ 関係構築困難

設計根拠：心理学「至適刺激水準理論」×コサイン類似度 をガウス曲線で変換。
現在の最適点 0.5 は仮説であり、実データ蓄積で継続検証予定。
